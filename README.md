# Minha Cafeteria

Um site de apresentação para uma cafeteria elegante, com foco em menu, produtos em destaque e atmosfera premium.

## Visão geral

Este projeto é uma landing page estática chamada **Lemos Coffee House**. Ele apresenta:

- Navegação fixa no topo
- Seção hero com chamada para ação
- Destaques de produtos especiais
- Sessões de confeitaria e café artesanal
- Animações de rolagem e estilo glassmorphism
- Videos

## Tecnologias usadas

- HTML
- CSS
- JavaScript

## Arquivos principais

- `index.html` — marcação da página
- `styles.css` — estilos personalizados adicionais
- `scripts.js` — comportamento simples de animação e intersecção

## Como usar

1. Abra `index.html` no navegador.
2. Navegue pela página para ver as seções, efeitos e layout responsivo.

## Personalização

- Edite o conteúdo em `index.html` para alterar textos, imagens e seções.
- Modifique `styles.css` para ajustar sombras, transições e estilos personalizados.
- Atualize `scripts.js` para adicionar mais efeitos de rolagem ou interações.

## Observações

Este projeto não requer servidor ou instalação. Basta abrir o arquivo `index.html` localmente para visualizar a página.
